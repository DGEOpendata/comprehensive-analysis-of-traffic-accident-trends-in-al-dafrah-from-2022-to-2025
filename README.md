## Traffic Accident Analysis in Al Dafrah

### Overview
This project provides a comprehensive analysis of traffic accident trends in Al Dafrah from 2022 to 2025 using the dataset 'Traffic Accident Reports in Al Dafrah Data for 2022 to 2025'. The analysis focuses on identifying patterns in accident occurrences, evaluating the impact of weather conditions, and predicting peak accident times.

### Prerequisites
- Python 3.x
- Pandas library
- Matplotlib library
- Jupyter Notebook (optional for interactive analysis)

### Getting Started
1. **Clone the Repository**:
   bash
   git clone https://github.com/your-repository/traffic-accident-analysis.git
   cd traffic-accident-analysis
   

2. **Install Required Libraries**:
   bash
   pip install pandas matplotlib
   

3. **Download the Dataset**:
   Ensure the dataset 'Traffic Accident-v3.0 (values) Al Dafrah 2022-2025.xlsx' is in the project directory.

4. **Run the Analysis**:
   Open the `analysis.py` script in your preferred Python environment and execute the code to generate insights and visualizations.

### Key Analyses
- **Annual Accident Trends**: Visualize the number of accidents per year to assess trends over time.
- **Weather Impact**: Analyze how different weather conditions affect accident rates.
- **Peak Accident Times**: Identify the hours of the day with the highest accident frequencies.
